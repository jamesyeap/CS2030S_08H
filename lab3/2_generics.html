      to[i] = from[i];

‚úèÔ∏è Exercise 5

write a generic class D<T> that contains a field that is an array of type T with 10 elements.

Instantiate that array in the constructor.

solution

class D<T> {
  T[] a;

  // constructor
  D() {
    @SuppressWarnings("unchecked")
    T[] tmp = (T[]) new Object[10];

    this.a = tmp;
  }
}
